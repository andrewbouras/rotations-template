# Clinical Rotations Command Center

Landing page for **The Clinical Rotations Command Center** — a Notion template for MS3/MS4 medical students.

**Live site:** https://andrewbouras.github.io/rotations-template/

**Product listing:** [Gumroad](#) ← update once listed

## What Is This?

A comprehensive Notion workspace for clinical rotations:
- 7 interconnected databases (Rotation Tracker, Patient Log, Shelf Planner, Procedure Log, Eval Tracker, Pimping Prep, Daily Reflections)
- 43 pimping questions + answers across all core rotations
- 50+ procedures pre-loaded by specialty
- COMLEX Level 2 focus areas for DO students
- Daily schedule templates for every rotation
- Complete ERAS prep hub

Built by an OMS-III who needed this during rotations.
